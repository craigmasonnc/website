I'm an enthusiast

My name is Craig Mason, and this is my personal website! I'm a developer, and I love tinkering with all kinds of programming languages. I built this website myself and set up an Ubuntu server in my home to host it. On this site you'll find some information about me as well as some demos of things I've been learning about.
